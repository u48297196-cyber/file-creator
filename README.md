# 🚀 FileCreator — Vibe-Coding Toolset

**FileCreator** — это комплексный набор инструментов для автоматизации разработки и обучения этичному хакингу. Проект объединяет функционал для аудита систем и быстрой генерации рабочего окружения.

## 🛠 Основные модули

*   **File Creator:** Мгновенная генерация файловых структур для Python, JS, C++ и других языков.
*   **Recon 3 (Разведка):** Модуль анализа заголовков и аудита доступности ресурсов (только для полигонов).
*   **System Monitor:** Контроль ресурсов ПК (нагрузка на CPU и GPU).
*   **Security Tools:** Генератор стойких паролей и учебный модуль шифрования.
*   **Resources:** Необходимые библиотеки и файлы (папку **не удалять!**).

## ⚠️ Дисклеймер (Важно)

Инструмент создан **исключительно для образования**. Используйте его только на своих машинах или учебных стендах (например, *scanme.nmap.org*). Автор не несет ответственности за ваши действия. Работайте легально!

## 📋 Инструкция по запуску

1. Установите зависимости:
   `pip install psutil GPUtil scapy requests colorama`
2. Перейдите в папку нужного инструмента.
3. Запустите: `python main.py` (или имя конкретного скрипта).
4. **Важно:** Не перемещайте файлы и **не удаляйте папку tests**, она нужна для проверки корректности работы модулей.
5. Результаты аудита сохраняются в `scan_report.txt` и качайте всю папку чтоб все работало!! чтоби скачать всю папку нажмите на code
6. и download zip но на етом все!!
